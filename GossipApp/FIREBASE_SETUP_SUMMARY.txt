📍 STEP 1: Create Firebase Project
──────────────────────────────────────────────────────────────────────────────
  1. Go to: https://console.firebase.google.com/
  2. Click: "Add project" or "Create a project"
  3. Name: "GossipIn" (or "gossipin")
  4. Disable Google Analytics
  5. Click: "Create project"
  ⏱ Time: ~2 minutes

📍 STEP 2: Enable Anonymous Authentication
──────────────────────────────────────────────────────────────────────────────
  1. Click: Authentication (left sidebar)
  2. Click: "Get started"
  3. Click: "Sign-in method" tab
  4. Click: "Anonymous" provider
  5. Toggle: "Enable" switch → Save
  ⏱ Time: ~1 minute

📍 STEP 3: Create Firestore Database
──────────────────────────────────────────────────────────────────────────────
  1. Click: Firestore Database (left sidebar)
  2. Click: "Create database"
  3. Choose: "Start in production mode"
  4. Select: Location (e.g., us-central1)
  5. Click: "Enable"
  ⏱ Time: ~2 minutes

📍 STEP 4: Add Android App & Download Config
──────────────────────────────────────────────────────────────────────────────
  1. Click: Project Settings (⚙ gear icon)
  2. Scroll: "Your apps" section
  3. Click: Android icon (</>)
  4. Enter Package: com.gossipin
  5. Enter Nickname: GossipIn
  6. Click: "Register app"
  7. DOWNLOAD: google-services.json
  8. Place in: Downloads folder
  ⏱ Time: ~2 minutes

                                🆘 HELP & SUPPORT
================================================================================

  Common Issues:

  ❌ "google-services.json not found"
     → Run: .\setup-google-services.bat

  ❌ "Firebase project not found"
     → Check .firebaserc file
     → Run: firebase login

  ❌ "Permission denied" in app
     → Run: firebase deploy --only firestore:rules

  ❌ Build errors
     → Run: cd android && gradlew clean && cd ..
